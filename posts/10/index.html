安装配置 traefik ingress

1. 简介

  • 理解Ingress

    简单的说，ingress就是从kubernetes集群外访问集群的入口，将用户的URL请求转
    发到不同的service上。Ingress相当于nginx、apache等负载均衡方向代理服务器，
    其中还包括规则定义，即URL的路由信息，路由信息得的刷新由Ingress controller
    来提供。

  • 理解Ingress Controller

    Ingress Controller 实质上可以理解为是个监视器，Ingress Controller 通过不断地跟
    kubernetes API 打交道，实时的感知后端 service、pod 等变化，比如新增和减少
    pod，service 增加与减少等；当得到这些变化信息后，Ingress Controller 再结合下
    文的 Ingress 生成配置，然后更新反向代理负载均衡器，并刷新其配置，达到服务
    发现的作用。
